## ✨ Features
- Search weather by city name 🌍  
- Displays:
  - Current temperature 🌡️  
  - City name 🏙️  
  - Humidity 💧  
  - Wind speed 🌬️  
- Dynamic weather icons (clouds, clear, rain, drizzle, mist)  
- Shows error message for invalid city ❌  

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- OpenWeather API
